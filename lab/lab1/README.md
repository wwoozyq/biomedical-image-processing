# 实验一：图像基础

## 实验目的

1. 理解数字图像的基本概念
2. 观察人眼视觉系统的特性（马赫带效应）
3. 理解灰度级量化对图像质量的影响

## 实验内容

### 1. 灰度渐变图生成

生成一个256×256像素的灰度图像，灰度值从左到右线性变化（0-255）。

**关键代码：**
```python
image = np.tile(np.arange(256, dtype=np.uint8), (256, 1))
```

**预期结果：**
- 图像尺寸：256×256
- 灰度值范围：0-255
- 视觉效果：从左到右由黑到白的平滑渐变

### 2. 马赫带效应观察

生成带状灰度图，每个带宽16像素，灰度值以16为步长递增。

**关键代码：**
```python
for i in range(16):
    gray_value = i * 16
    image_bands[:, i*16:(i+1)*16] = gray_value
```

**马赫带效应：**
- 在相邻灰度带的边界处，人眼会感知到不存在的亮带和暗带
- 这是人眼视觉系统的侧抑制现象导致的

### 3. 灰度级量化

将图像量化到不同的灰度级别（256级、64级、16级），观察量化对图像质量的影响。

**量化公式：**
```python
levels = 2 ** n
step = 256 / levels
y = np.floor(x / step) * step
```

**实验结果：**
- 256级（2^8）：与原图几乎无差别
- 64级（2^6）：轻微的阶梯效应
- 16级（2^4）：明显的假轮廓现象
