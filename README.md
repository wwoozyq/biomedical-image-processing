# 生物医学图像处理课程项目

本项目包含生物医学图像处理课程的实验代码和笔记。

## 项目结构

```
.
├── README.md           # 项目说明
├── lab/               # 实验代码
│   └── lab1/         # 实验一
│       ├── lab1.ipynb    # 实验代码
│       ├── lab1.npy      # 实验数据
│       └── README.md     # 实验说明
└── notebook.md        # 课程笔记
```

## 环境要求

- Python 3.x
- NumPy
- Matplotlib
- Jupyter Notebook

## 快速开始

1. 安装依赖：
```bash
pip install numpy matplotlib jupyter
```

2. 启动Jupyter Notebook：
```bash
jupyter notebook
```

3. 打开 `lab/lab1/lab1.ipynb` 开始实验
